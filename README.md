# FIFAWC_ANALYSIS
1️. Project Title / Headline:
FIFA World Cup 2022 Analytics Dashboard — Performance, Trends, and Insights using Power BI

2️. Short Description / Purpose:
This project is an interactive Power BI dashboard built to analyze historical FIFA World Cup data along with the 2022 tournament. The goal is to explore trends in team performance, goal statistics, and host country outcomes using data visualization and business intelligence techniques.
The dashboard transforms raw match data into meaningful insights that help answer key questions such as how teams performed historically, how host nations usually perform in World Cups, and which teams overperformed or underperformed in the 2022 tournament.
The project demonstrates practical skills in data cleaning, data modeling, DAX calculations, and dashboard design.

3️. Tech Stack:
The dashboard was built using the following tools and technologies:
Power BI Desktop – Data modeling and dashboard development
Power Query – Data cleaning and transformation
DAX (Data Analysis Expressions) – Measures and calculated columns
Data Modeling – Relationships and star schema design
Microsoft Excel / CSV datasets – Source data files

4️. Data Source:
The dataset contains historical FIFA World Cup data including:
World Cup match results
Goals scored by teams
Tournament hosts and winners
Squad and group information for the 2022 World Cup
Historical international match records
The data was compiled from publicly available football datasets commonly used for sports analytics and visualization projects.

5️. Highlights / Features:
-> Business Problem:
Football tournaments generate large amounts of data, but it is difficult to quickly understand performance trends, team dominance, or historical patterns without proper analysis.
The challenge was to convert raw football data into an interactive analytics dashboard that helps users explore insights about teams, goals, and tournament performance.

-> Goal of the Dashboard:
The main objectives of the dashboard were:
Analyze historical goal scoring trends
Compare team performances across tournaments
Understand host nation performance patterns
Identify overachieving and underperforming teams in the 2022 World Cup
Provide an interactive interface for exploring football data

->Walkthrough of Key Visuals:
KPI Cards

The dashboard starts with key indicators showing:
Total goals scored across matches
Total matches played
Average goals per match
Number of participating teams
These metrics provide a quick overview of the dataset and tournament statistics.

->Goal Scoring Trend:
A time-series visualization shows how goal scoring has evolved across World Cups.
This helps identify whether modern tournaments produce more goals compared to earlier competitions.

->Team Goal Comparison:
A bar chart ranks teams based on total goals scored, highlighting historically dominant teams such as Brazil, Germany, and Argentina.
This visualization helps quickly identify which countries have been the most consistent attacking teams in World Cup history.

->Host Country Performance Analysis:
This section analyzes how host nations perform in World Cups.
The dashboard compares host countries based on:
Final tournament position
Total goals scored
Tournament outcomes
One key observation is that host nations often perform strongly due to home advantage, with many reaching the later stages of the tournament.
However, Qatar (2022) stands out as an exception, becoming one of the few host nations eliminated during the group stage.

->Overachievers vs Underperformers (2022):
This analysis compares historical team strength with actual results in the 2022 tournament.
Teams with performance above expectations are highlighted as overachievers, while those performing below historical standards are categorized as underperformers.
For example:
Morocco emerged as one of the biggest overachievers, reaching the semi-finals despite a modest historical record.
Germany underperformed relative to its strong World Cup history.

-> Business Impact and Insights:
This dashboard demonstrates how data visualization can help uncover meaningful insights from sports data.
Some key insights include:
Historically dominant teams continue to influence goal scoring trends.
Host nations often benefit from playing at home, although exceptions exist.
Data-driven comparisons reveal surprising tournament outcomes such as unexpected overperforming teams.

Beyond football analysis, the project highlights the importance of data storytelling, interactive dashboards, and analytical thinking, which are essential skills in modern data analytics roles.
